# AIX Bond Page Assets

This folder contains the local files required by `aix-bond.blade.php`, copied from `https://aixfincon.com/promoted-product/aix-bond/`.

Laravel placement:
- Put this folder at `public/asset/aix-bond`.
- Put `aix-bond.blade.php` at `resources/views/aix-bond.blade.php`.
- The Blade page uses helpers like `{{ asset('asset/aix-bond/...') }}`.
